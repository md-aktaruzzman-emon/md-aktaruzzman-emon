### 🌐 Executive Profile
I am a Computer Science undergraduate focused on the intersection of **systems engineering** and **cybersecurity**. My engineering philosophy revolves around creating robust, low-overhead tools in C/C++ and Java, while leveraging Python as my primary force-multiplier for automation, security auditing, and rapid prototyping. 

With a strong foundation in data structures, algorithms, and software design principles, I build security-minded applications, analyze software vulnerabilities, and contribute to the open-source defensive security community.
