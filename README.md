### 🌐 Executive Profile
I am a Computer Science undergraduate specializing in **low-level systems development**, **database architecture**, and **defensive cybersecurity**. I bridge the gap between hardware-level performance and secure application design. By leveraging C/C++ for low-latency engineering, Java for enterprise-grade Object-Oriented patterns, and Python for rapid exploit automation and security scripts, I build applications that are optimized, scalable, and resilient.

With a deep academic focus on Data Structures and Algorithms (DSA), I specialize in writing clean, mathematically sound algorithms and robust, secure code.

### ⚡ Architectural & Programming Pillars

#### 🛠️ C & C++ Systems Engineering
*   **Low-Level Mastery:** Solid command of pointers, manual memory management, RAII (Resource Acquisition Is Initialization), and memory layouts.
*   **Performance Optimization:** Writing high-efficiency algorithms, reducing CPU cycles, and creating close-to-metal tooling.
*   **Compilation & Debugging:** Competent in using GCC/Clang, GDB, Valgrind for memory leak profiling, and building cross-platform binaries.

#### 🏗️ Object-Oriented Programming (OOP) & Design
*   **System Abstraction:** Strong application of SOLID principles, encapsulation, polymorphism, inheritance, and architectural design patterns.
*   **Enterprise Tooling:** Constructing modular, reusable codebases in Java and C++ that scale cleanly without regressions.

#### 📊 Data Structures & Algorithms (DSA)
*   **Complexity Analysis:** Rigorous evaluation of space/time complexities ($O(n)$, $O(\log n)$, etc.) to ensure optimal performance.
*   **Advanced Structures:** Implementation and utilization of trees, graphs, hash tables, heaps, and advanced sorting/searching methods.
*   **Dynamic Problem Solving:** Tackling algorithmic challenges using dynamic programming, greedy strategies, and divide-and-conquer paradigms.

#### 🗄️ Database Architecture
*   **Relational Database Design:** Normalization (1NF, 2NF, 3NF), entity-relationship modeling, and maintaining foreign key constraints.
*   **Query Optimization:** Writing efficient SQL queries, indexing, and managing transactions.
*   **Tech Stack:** Designing backend logic with **PostgreSQL**, **MySQL**, and **SQLite**.
